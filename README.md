# VabHub Frontend · Web 前端 (Vue 3)

[![Build](https://img.shields.io/badge/build-passing-brightgreen.svg)](#)
[![Node](https://img.shields.io/badge/node-%3E=20-orange.svg)](#)

Web 前端（Vue 3 + TypeScript）：**站点管理（含优先级排序）、规则配置、订阅与任务、纠错重整、实时日志、设置**。

## 本地开发
```bash
pnpm i   # 或 npm i / yarn
pnpm dev
```
**环境变量**
```
# .env.local
VITE_API_BASE=http://localhost:8081
```
